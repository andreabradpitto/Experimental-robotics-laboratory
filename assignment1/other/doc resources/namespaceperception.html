def perception.perception ( )

Simulate the sensors capturing user's <<play>> requests and subsequent pointing gestures.

The user is assumed to be aware of the robot's current state (e.g. via LEDs on the robot itself) and asks to <<play>> only if MiRo is not sleeping or already playing, i.e. it is in Normal state
